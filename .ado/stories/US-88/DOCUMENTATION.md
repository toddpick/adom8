## API Documentation

### Notification Manager Methods

- `addNotification(notification)`: Add a new notification
  - `notification` object properties:
    - `emoji`: Emoji representing notification type
    - `title`: Notification title
    - `message`: Notification message
    - `type`: Notification type (completed/failed/delegated)
    - `agent`: Agent name
    - `workItemId`: Associated work item ID

- `toggleSetting(key)`: Toggle notification settings
  - Supported keys: `toast`, `browser`, `sound`

- `setVolume(value)`: Set notification volume (0-100)

### Notification Types

- Toast Notifications: Slide-in alerts at bottom-right
- Browser Notifications: Desktop alerts when tab unfocused
- Sound Alerts: Web Audio API chimes/tones

// Add a notification

  emoji: '✅',
  title: 'Coding Agent Completed',
  message: 'US-88: Story Planning',
  type: 'completed',
  agent: 'Coding',
  workItemId: 88

// Toggle notification settings

// Set volume
