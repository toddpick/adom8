## API Documentation

### Filter and Search API

#### Methods

- `handleSearchFilter()`: Triggered on search input
  - Updates `currentSearchText`
  - Persists search text in localStorage
  - Calls `renderFilteredStories()`

- `handleStatusFilter(status: string)`: Triggered on status chip click
  - Updates `currentStatusFilter`
  - Persists status filter in localStorage
  - Updates active filter chip styling
  - Calls `renderFilteredStories()`

- `renderFilteredStories()`: Core filtering logic
  - Filters stories based on search text and status
  - Updates result count display
  - Renders matching story cards

#### Filter State Persistence

